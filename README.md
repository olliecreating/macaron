# Macaron

**A playful neo-brutalist theme for Zen Browser.**


<img width="3000" height="1505" alt="Macharon_Zen" src="https://github.com/user-attachments/assets/08be252f-6bca-4dde-9429-52f676ac854b" />
<br>

Macaron turns Zen Browser into something that feels less like software and more like a tiny collectible device sitting on your desktop.

Think:

- chunky borders  
- tactile shadows  
- candy-striped loading animations  
- playful checkerboard blank tabs  
- squircle macOS controls  
- retro gadget energy  

Built for people who think browsers should feel fun again.

---

## Local Options

- Accent presets: Strawberry, Pistachio, Blueberry, Vanilla, Matcha.
- Shadow intensity: None, Soft, Classic, Chunky.
- Browser spacing: Macaron, Tight, Marginless.
- Macaron URL bar popup styling.
- Doodle pack: Coffee, Star, Flower, None.
- Reduced motion toggle.

## Installation

Install easily through **Sine**, a modern mod manager for Firefox-based browsers.

### 1. Install Sine

Download and install Sine.

### 2. Add Macaron

Search for **Macaron**

**or**

paste this into custom mod input:

```bash
SehajveerSingh2005/macaron
```

### 3. Apply Theme

Enable Macaron from your installed mods.

Done.

No more manually digging through `about:profiles` like you're entering forbidden ruins beneath Firefox.

## Philosophy

Most browser themes try to disappear.

Macaron does the opposite.

It leans into playful imperfection.
It feels tactile.
It feels physical.
It feels like software you could hold.

Somewhere between:

* retro handheld consoles
* stationery design
* neo-brutalist websites
* toy keyboards
* weird internet experiments

---

## Why I Built This

Because every browser started feeling like the same gray rectangle.

And that felt... boring.
